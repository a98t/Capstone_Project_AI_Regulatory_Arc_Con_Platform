# 🎓 Capstone Project Topic

## **Development of an AI-Powered Regulatory Intelligence System for the Construction Industry Using RAG and Multi-Agent Architecture**

---

# 💡 Project Description

This project focuses on designing and developing an intelligent system that enhances traditional regulatory databases by integrating:

- Retrieval-Augmented Generation (RAG)
- AI agents for task automation
- Model Context Protocol (MCP) for structured context handling

The system enables:
- Semantic search of regulations
- Automated compliance analysis
- Contextual recommendations for engineering and architectural projects

# 🧩 System Overview (DEREK-INFO 2.0)

## 🔍 1. RAG-Based Search Engine
- Upload regulatory documents (PDFs)
- Convert into embeddings
- Perform semantic search instead of keyword-based search

**Example Query:**
> "Find all fire safety rules for a 5-floor residential building"

---

## 🤖 2. AI Agents (Core Innovation)

### Agents in the system:
- **Search Agent** → retrieves relevant norms
- **Compliance Agent** → checks project against regulations
- **Explanation Agent** → simplifies complex rules
- **Update Agent** → tracks regulatory updates

---

## 🧠 3. Model Context Protocol (MCP)

Manages structured context such as:
- Project type
- Location
- Building parameters

Ensures accurate and context-aware responses.

---

## 📊 4. Smart Outputs

- Compliance reports
- Risk identification
- Suggested corrections

---

# 🚀 Example Use Case
